Galeria de Componentes de UI

Está em dúvidas sobre as propriedades dos componentes? Veja esse guia

Princípios Fundamentais

Uma Preocupação, Um Prop

Estado: state OU type (nunca color).

Boolean vs Enum

Boolean para binário (isX, hasY). Enum para múltiplas opções (intent, size).

Nomes Claros

isDisabled, hasIcon, isOn - nomes claros e Figma-friendly.

Semântica Primeiro

Prefira semântica quando há significado. Estética só quando necessário.
